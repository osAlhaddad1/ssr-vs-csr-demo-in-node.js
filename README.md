### SSR vs CSR

this is a demostation between the speed of ssr and csr

to run the demo, run the commands:

`cd server`

and then:

`node server-csr.js`

`node server-ssr.js`


on the browser open the dev tool and go to the network tab, and there for the network speed select `3G` instead of `No throttling` and the relaod the website and see the difference in speed
